Welcome to the Puzzles RPG Help Page!

To get started, make your way to the home page by clicking on the "Puzzles RPG" heading at the top of each page. Once there, you can select pre-existing characters to modify or delete them, or create a new character using the buttons that indicate each activity.

When looking at a character page, you can view the attributes, inventory, and skills of the character. From this page you can also select the desired three skills. You can only select 1 skill per row grouping, so pick carefully!

To edit your account, settings, or if you have comments or concerns, see the links in the footer.
